### 3. دوال RPC محسّنة ✅

**تم تحسين:**

#### `approve_order()`
- إضافة فحص صلاحيات المسؤول
- التحقق من حالة الطلب
- تحديث المخزون بشكل صحيح
- معالجة أخطاء محسّنة

#### `reject_order()`
- إضافة فحص صلاحيات المسؤول
- التحقق من حالة الطلب
- إعادة المخزون إلى متاح
- دعم ملاحظات المسؤول

---

### 4. دوال جديدة للصيانة ✅

#### `check_data_integrity()`
فحص سلامة البيانات:
- الطلبات اليتيمة (بدون مستخدم/منتج)
- المعاملات اليتيمة (بدون مستخدم)
- المخزون غير الصحيح (بدون منتج)

**نتيجة الفحص:**
```json
{
  "orphaned_orders": 0,
  "orphaned_transactions": 0,
  "invalid_inventory": 0,
  "has_issues": false
}
```
✅ لا توجد مشاكل في سلامة البيانات!

#### `get_system_stats()`
إحصائيات شاملة للنظام:
```json
{
  "total_users": 4,
  "active_users": 4,
  "verified_users": 2,
  "total_products": 0,
  "available_inventory": 0,
  "total_orders": 0,
  "pending_orders": 0,
  "approved_orders": 0,
  "total_wallet_balance": 2000,
  "pending_topups": 0
}
```
